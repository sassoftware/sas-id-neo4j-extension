To install the Neo4j nodes in SAS Intelligent Decisioning, you must first copy the installation resources to a Linux server. This server can be either the SAS Viya server itself or a jump server with access to the SAS Viya environment.


The server must also have the SAS Viya CLI installed, as it is required to deploy the nodes into SAS Intelligent Decisioning.


You can copy the installation resources from the Git repository manually, or you can use the provided automated installation script.
